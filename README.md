# 📊 Super Store Sales Analytics & 15-Day Sales Forecast Dashboard

### An interactive Power BI dashboard built to analyze retail sales performance, customer behavior, shipping trends, regional insights, and forecast future sales using historical data from 2019–2021.

The dashboard transforms raw retail transaction data into actionable business intelligence, enabling stakeholders to monitor KPIs, identify growth opportunities, optimize operations, and make data-driven decisions.

---

# 2. 📌 Short Description / Purpose

The **Super Store Sales Analytics & 15-Day Sales Forecast Dashboard** is a comprehensive Power BI solution designed to analyze sales, profit, customer segments, product categories, and regional performance across the United States. Additionally, the dashboard incorporates predictive analytics to forecast future sales trends based on historical data.

This tool is intended for **business analysts, sales managers, retailers, supply chain teams, and decision-makers** seeking actionable insights to improve profitability and operational efficiency.

---

# 3. 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Main data visualization and dashboard development platform.
* 📂 **Microsoft Excel** – Used as the primary data source for storing retail transaction data.
* 🔄 **Power Query** – Performed data cleaning, transformation, and preprocessing (ETL).
* 🧠 **DAX (Data Analysis Expressions)** – Used for creating calculated measures, KPIs, and business logic.
* 🤖 **AI Tools** – Assisted in dashboard design, insight generation, and analytical storytelling.
* 📝 **Data Modeling** – Established relationships between tables for efficient analysis and cross-filtering.
* 🗺 **Bing Maps** – Enabled geographic visualization of state-wise sales and profit.
* 📈 **Power BI Forecasting** – Implemented time-series forecasting to predict future sales.
* 📁 **File Formats** – `.xlsx` for data storage, `.pbix` for development, and `.png` for dashboard previews.

---

# 4. 📂 Data Source

**Source:** Super Store Sales Dataset (Retail Dataset)

The dataset contains retail transaction records of a superstore operating across multiple regions of the United States.

The dataset includes:

* Order ID
* Order Date
* Ship Date
* Ship Mode
* Customer Segment
* Region
* State
* Product Category
* Product Sub-Category
* Sales
* Profit
* Quantity

The data spans **2019–2021** and supports descriptive analytics, trend analysis, and predictive forecasting.

# 5. ✨ Features / Highlights

## 🔍 Business Problem

Retail businesses generate massive amounts of transactional data every day. However, extracting meaningful insights from raw data is challenging without an analytical system.

Key business questions include:

* Which regions generate the highest sales and profit?
* Which product categories perform best?
* Which customer segments contribute most to revenue?
* Which shipping modes are preferred by customers?
* How do sales and profits change over time?
* What will future sales look like?

These questions are difficult to answer efficiently using spreadsheets or static reports.

---

## 🎯 Goal of the Dashboard

To deliver an interactive analytical solution that:

* Monitors sales performance in real time.
* Tracks profit and revenue trends.
* Identifies high-performing products and regions.
* Analyzes customer purchasing behavior.
* Supports inventory and supply chain planning.
* Predicts future sales using forecasting techniques.
* Enables data-driven decision-making.
